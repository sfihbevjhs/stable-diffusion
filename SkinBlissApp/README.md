# Skin Bliss - Mobile Skincare App

A beautiful mobile application for skincare enthusiasts built with React Native and Expo.

## Features

- **Beautiful Onboarding Screen**: Elegant design with gradient overlays and smooth animations
- **Social Authentication**: Sign in with Google or Apple
- **Responsive Design**: Works seamlessly on both iOS and Android
- **Modern UI**: Clean, minimalist interface with premium aesthetics

## Getting Started

### Prerequisites

- Node.js (14 or later)
- npm or yarn
- Expo CLI

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd SkinBlissApp
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

### Running the App

- **Web**: `npm run web`
- **iOS Simulator**: `npm run ios` (requires macOS)
- **Android Emulator**: `npm run android`
- **Physical Device**: Scan QR code with Expo Go app
